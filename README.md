Often old versions of splunk are required for different reasons such as legacy OS'es. Sometimes old splunk versions are the only nice way to get data from these hosts.

This list of urls aims to help preserve and make these these older versions accessable. 

Note: Even though these are all hosted by splunk themselves they will likely NOT be supported! So dont try and log a ticket against a version that was released 10 years ago!

More info on their supported versions here : https://www.splunk.com/en_us/legal/splunk-software-support-policy.html
